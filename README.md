- React

- Styled Components
- Swiper js
- React Transition Group
- Smooth Scrollbar
- React Icons
- React Router Dom
- More...

## Getting Started

After getting the starter files, you need to go the file directory and run

```shell
npm install
```

and after that start the live server.

```shell
npm start
```
